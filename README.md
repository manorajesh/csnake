# CSnake

CSnake is snake, in C.

## Features

- snake

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/manorajesh/csnake.git
   cd csnake
   ```

2. Compile the game:

   ```sh
   make
   ```

3. Run the game:
   ```sh
   ./csnake
   ```

## Controls

- **WASD**: Move the snake up, down, left, or right
- **Q**: Quit the game
- **+**: To cheat
